General

1-4 Switch mouse mode
shift-drag Box-select
ctrl-drag Time-select
space-drag Pan
f Zoom into selection
z/0 Reset zoom and pan
v Highlight VSync
m Mark current selection
h/? Show help

Navigation

w/s (or Ctrl +/-) Zoom in/out (+shift: faster)
a/d Pan left/right (+shift: faster)
q/e Pan up/down (+shift: faster)

Mouse Controls

click Select event
ctrl-mousewheel Zoom in/out

Modes

Select Mode

drag Box select

Pan mode

drag Pan the view

Zoom mode

drag Zoom in/out by dragging up/down

Timing mode

drag Create or move markers
